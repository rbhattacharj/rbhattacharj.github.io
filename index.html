Research Interests

I'm broadly interested in theoretical computer science and machine learning. I leverage tools from randomized algorithms, linear algebra, optimization and machine learning to design algorithms which can proccess large amounts of data efficiently. I'm also interested in understanding the computational and statistical tradeoffs for learning algorithms.

I'm currently working on algorithms which can approximate the eigenspectrum of a matrix in sublinear time. Previously, I've worked on problems related to online learning. You can check out some of my publications below.
